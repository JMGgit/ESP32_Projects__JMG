In mode OFF:
- Button 1 pressed 3s: change RGB LED order
- Button 2 pressed 3s: change LED connections order (daisy chain or lines)
- Button 3 pressed 3s: enable / disable startup mode
- Button up pressed 3s: disable / enable FOTA cyclic check (Flash Over The Air)
- Button left pressed: display FOTA state
- Button down pressed: SW reset
